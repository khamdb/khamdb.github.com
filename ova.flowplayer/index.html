OVA for Flowplayer - Version 1.2.0

Welcome to the v1.2.0 release of the Open Video Ads streamer for Flowplayer.

This package contains the binary (SWF) files required to deploy the plugin with Flowplayer 3.2.7 +.

The package also contains the source files and ant build files.

Several examples are also provided to help illustrate the various configurations for the plugin.

Further documentation can be found on the detailed plugin documentation pages at the google code repository www.openvideoads.org
