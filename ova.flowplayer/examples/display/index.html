OVA for Flowplayer - Configuring the Display

  1. Default Linear Mode Configurations (Margins and Control Bar Behaviour)

    • VPAID Linear ad - 'never hide' control bar (always hide) (fullscreen hide) (none)
    • VPAID Linear ad going to Non-Linear - 'never hide' control bar (always hide) (fullscreen hide) (none)
  2. Default Non-Linear Mode Configurations (Margins and Control Bar Behaviour)
    • Static overlay - 'never hide' control bar (always hide) (fullscreen hide) (none)
    • VPAID overlay - 'never hide' control bar (always hide) (fullscreen hide) (none)
  3. Setting Custom Margins
    • Modifying the bottom margin - all display modes - all ad types
    • Modifying the bottom margin - normal display - all ad types
    • Modifying the bottom margin - fullscreen - all ad types
  4. Declaring Custom Control Bar Settings
    • Hard coding the control-bar height - linear and non-linear ads
    • Hiding the control bar during linear ad playback - all types (stream) (VPAID)
    • Showing the control bar during linear ad playback - all linear types (stream) (VPAID)
    • Stopping OVA from changing the control bar during linear ad playback - always (stream) (VPAID)
    • Fine grained control bar manipulation
      • Disable all controls - all linear ad types (Linear streams) (Linear VPAID)
      • Selective enabling - all linear ad types (Linear streams) (Linear VPAID)
