OVA for Flowplayer - Worked Examples

A number of examples have been included with this release to showcase the available range of configuration options. All of these examples require Flowplayer 3.2.7 or higher.

Broadly speaking, the examples have been organised into the following groupings:

  • VAST/VPAID Tag Validator
  • Flowplayer Setup
    • Minimal JSON setup
    • Long form JSON setup
    • Turning OVA debug output off
    • Plugins, playlists and general options
  • Ad Types
    • Illustrating the Supported Ad Formats
    • Delivering RTMP Linear Ads
    • Non-Linear Ads (Overlays)
    • Running Interactive (VPAID) Ads
    • Supporting Companions
    • Delivering "Ad Stories"
    • Ads on Live Streams
    • Survey Support
  • Ad Tags, Calls and Servers
    • Delayed and On-Demand Ad Loading
    • Configuring the Ad Servers
    • Using OVA Variables and Transformers with Ad Tags
    • VAST Wrapper
    • Direct VAST Data Injection
    • Supporting the VAST Survey Tag
    • Special Ad Tag Test Cases
    • Custom Ad Delivery Options
  • Visual Elements
    • Configuring the Player Display
    • Ad and Click-Through Notices
    • Skip Linear Ad Options
  • HTML5 and Javascript Support
    • The Javascript API
    • HTML 5 support
  • Analytics
    • Tracking with Google Analytics
    • Supporting Custom Event Logging
  • General Test Cases
    • Specifying a Start Time (ad or show stream)
    • Supporting streams and ads without a duration specified

A number of the examples use an OpenX installation that we are running at Open Video Ads. To gain more insight into the zone/banner setup that we have in place for the examples, please read this short guide.
